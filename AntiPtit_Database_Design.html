SƠ ĐỒ CÁC BẢNG DATABASE - BLOGHUB

Thiết kế cấu trúc các Collections chi tiết cho hệ quản trị cơ sở dữ liệu MongoDB

Khóa chính (PK) Khóa ngoại (FK) Ràng buộc Unique Chỉ mục Index

users

Collection
Tên trường Kiểu dữ liệu Khóa/Ràng buộc
_id ObjectId PK
username String UQ
email String UQ
password_hash String -
display_name String -
avatar String -
bio String -
role String Default: 'user'
status String Default: 'active'
created_at Date Default: Date.now

posts

Collection
Tên trường Kiểu dữ liệu Khóa/Ràng buộc
_id ObjectId PK
title String IDX
slug String UQ
content String IDX
summary String -
cover_image String -
status String Default: 'published'
tags Array [String] IDX
author_id ObjectId FK (users)
likes Array [ObjectId] Mảng tham chiếu users
view_count Number Default: 0
created_at Date Date.now
updated_at Date Date.now

comments

Collection
Tên trường Kiểu dữ liệu Khóa/Ràng buộc
_id ObjectId PK
post_id ObjectId FK (posts)
user_id ObjectId FK (users)
content String Max length: 1000
status String Default: 'visible'
created_at Date Date.now

follows

Collection
Tên trường Kiểu dữ liệu Khóa/Ràng buộc
_id ObjectId PK
follower_id ObjectId FK (users)
following_id ObjectId FK (users)
created_at Date Date.now

reports

Collection
Tên trường Kiểu dữ liệu Khóa/Ràng buộc
_id ObjectId PK
reporter_id ObjectId FK (users)
target_type String Giá trị: 'post' hoặc 'comment'
target_id ObjectId ID của đối tượng bị báo cáo (Liên kết động)
reason String -
status String Default: 'pending'
created_at Date Date.now
